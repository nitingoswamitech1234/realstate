# Genesierealty - React + Vite + Tailwind Starter (Full)

This is a starter template matching the homepage layout you provided:
- Merged transparent header over a video hero banner
- Home page with Latest Projects, Amenities, Testimonials
- Placeholder About / Projects / Contact pages
- React Router + Tailwind

## Quick start

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

## Notes
- Replace `/public/assets/banner.mp4` with a real MP4 video for the hero.
- Replace images in `/public/assets/` with real photos.
- Tailwind is already configured in `tailwind.config.cjs`.
